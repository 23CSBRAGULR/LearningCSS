This sub-topic comes under the topic "CSS Box Model"

Every element in CSS is stored in Box layout. First the element itself, then the padding, then the borders, lastly the margins

Padding

Padding lets us edit the dimensions of the element itself by scaling the spaces on top, bottom, left, right side of the element

Borders

Borders are used for bordering the element where the style, color, width are customizable

Margin

Margin lets us adjust the space between the outer border of the element and the element itself, on all four sides
